# Smart Leads Dashboard 

A full-stack MERN Lead Management Dashboard built using:

- React + TypeScript
- Tailwind CSS
- Node.js + Express
- MongoDB
- JWT Authentication
- Recharts
- Vercel + Render Deployment

---

# Features 

## Authentication
- User Registration
- User Login
- JWT Authentication
- Protected Routes

## Lead Management
- Add Lead
- Edit Lead
- Delete Lead
- Search Leads
- Filter by Status
- Pagination

## Dashboard
- Statistics Cards
- Charts using Recharts
- CSV Export
- Toast Notifications
- Dark Mode

## Role-Based Access
- Admin can delete leads
- Sales users cannot delete leads
